# Интернет магазин с корзиной

Простой шаблон интернет магазина с функциональностью корзины.

## Структура проекта

- index.html - основная страница
- src/css/style.css - стили
- src/js/main.js - основной JavaScript код
- src/images/ - изображения товаров

## Функциональность

- Отображение списка товаров
- Добавление товаров в корзину
- Просмотр корзины
- Изменение количества товаров
- Удаление товаров из корзины
- Подсчет общей стоимости

## Использование

1. Создайте папку src и подпапки css, js, images
2. Создайте файлы index.html, src/css/style.css, src/js/main.js
3. Добавьте изображения товаров в папку src/images
4. Откройте index.html в браузере
